# Reinforcement Learning Assignments

A collection of assignments exploring core reinforcement learning concepts through hands-on implementation and experimentation, using Python and the [Gymnasium](https://gymnasium.farama.org/) environment library.

**Author:** Chris Hirschbrich

---

## Projects

### [Frozen Lake - Monte Carlo & TD Control](./frozen_lake)
Tabular RL methods applied to a classic grid-world environment, with experiments across varying levels of stochasticity.

### [Lunar Lander - Value Function Approximation](./lunar_lander_approximation)
Linear function approximation with tile coding applied to a continuous state space environment.

---

## Setup

```bash
pip install gymnasium
```

A patched version of `gym-classics` is also required. See the installation instructions within each notebook for details.
